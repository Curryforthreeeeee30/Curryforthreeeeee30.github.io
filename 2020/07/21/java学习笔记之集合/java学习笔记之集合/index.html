java集合的概念

定义：如果一个Java对象能够在内部持有其他Java对象，并对外提供访问接口，那么这个对象就是集合。
有各种各样的集合，比如：
1.数组
2.元素不重复的集合
3.可变大小的顺序链表
。。。。。。
java.util自带集合类，主要提供三种集合类：
1.list: 有序列表的集合
2.Set: 没有重复元素的集合
3.Map: “key-value”类型的集合
今天主要记录的是Set，其中最著名的是Hash Set。
